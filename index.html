🚀 安装到桌面
# 1. 创建目录
mkdir -p ~/.codex/pets/my-cat
 
# 2. 放入下载的两个文件
mv spritesheet.webp pet.json ~/.codex/pets/my-cat/
 
# 3. 启动桌面宠物
npx petdex install desktop
npx petdex hooks install
npx petdex desktop start

在 Codex 中 Settings → Appearance → Pets 选择 my-cat ✦
