🚀 安装到桌面（Petdex + Codex）
# 1. 创建宠物目录
mkdir -p ~/.codex/pets/my-cat
 
# 2. 将下载的两个文件放入该目录
mv spritesheet.webp ~/.codex/pets/my-cat/
mv pet.json ~/.codex/pets/my-cat/
 
# 3. 安装 Petdex 桌面应用
npx petdex install desktop
npx petdex hooks install
npx petdex desktop start

然后在 Codex 中 Settings → Appearance → Pets 选择 my-cat ✦
或直接用 npx petdex toggle 唤醒桌面宠物
